# Employee Management System

## 📌 Overview

The Employee Management System is a Python-based console application that helps manage employee records efficiently. The project allows users to perform CRUD (Create, Read, Update, Delete) operations while storing data permanently using a JSON file.

This project was developed to strengthen my understanding of Python programming, file handling, JSON integration, and data management concepts.

---

## ✨ Features

* ➕ Add New Employee
* 📋 View All Employees
* 🔍 Search Employee by ID
* ✏️ Update Employee Details
* ❌ Delete Employee Records
* 💾 Store Data in JSON File
* 🔄 Automatic Data Persistence
* 🖥️ Menu-Driven Interface
